PYTHON PRECISA DESSES PACOTES A DEPENDER DO SISTEMA

pip install psycopg2-binary
pip install prettytable
ou
pip3 install psycopg2-binary
pip3 install prettytable


PARA RODAR O PROGRAMA
python3 main.py

make docker-run
